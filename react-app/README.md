# 🏥 MEDINOVA - Healthcare Platform

> AI-powered full-stack healthcare platform connecting patients with doctors seamlessly.

## 🚀 Tech Stack
- **Frontend**: React 18, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **AI/ML**: TensorFlow, Dialogflow
- **Blockchain**: Ethereum Smart Contracts
- **Payments**: Stripe API

## 🛠️ Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm start
```
