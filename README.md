## 🧠 About The Project

**MEDINOVA** is a full-stack, AI-powered healthcare web application that bridges the gap between patients and medical professionals. It combines machine learning, blockchain, facial recognition, and natural language processing into a seamless platform covering the entire patient journey from first symptom to secured medical record.

> 💡 *"Every patient deserves professional medical guidance, instantly and securely — no matter where they are."*

---

## ✨ Key Features

| Feature | Description |
|--------|-------------|
| 🤖 **AI Symptom Detection & Chatbot** | Dialogflow-powered bot detects symptoms and recommends the right specialist |
| 📍 **Location-Based Doctor Search** | Find the nearest doctor in real time using Google Maps API |
| 📅 **Smart Appointment Booking** | Book with nearest or earliest available doctor via chatbot |
| 🔗 **Blockchain Medical Records** | Immutable patient data stored on Ethereum smart contracts |
| 🫁 **X-Ray Lung Diagnosis** | TensorFlow model detects COVID-19, Tuberculosis, Pneumonia |
| 🧠 **Mental Health Test** | Guided quiz with personalized article suggestions |
| 🪪 **Doctor Identity Verification** | National card ID verified via Nanonets AI OCR API |
| 🤳 **Multi-Mode Authentication** | Face ID, LinkedIn OAuth, and username/password login |
| 🛒 **Paramedical E-Shop** | ML-based personalized product recommendations |
| 💳 **Integrated Payments** | Stripe API for doctor subscriptions and patient payments |
